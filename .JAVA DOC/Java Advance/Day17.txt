How to add git hub project to Eclipse?
'''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''
1.create github repository
2.Start Eclipse
3.Goto Perspective->Git repositories and click on Add Git Repo
4.Create a project in Eclipse
5.Do aright click on project >Teams >Share >add to git repo
6.then again right click on project then click on commit
7.then drag the project into below box
8.then do commit and push
9.then write user name and password( if it says like not authorized then generate token by go to the github setting>developer tools>check repo>generate token and in place of password give token number)
10.then the project succesfully pushed in the github repository
